📚 FAQs

📌 Project Overview

This project is a Frequently Asked Questions (FAQs) page that features an interactive accordion-style layout using jQuery UI. Users can click on questions to expand or collapse the answers with smooth animations. This project was developed as a college assignment to practice JavaScript, jQuery UI, and CSS styling.

🚀 Features

🔽 Accordion-style FAQs: Users can expand and collapse sections interactively.

🎨 Animated UI: Smooth opening and closing of questions.

🖱️ Click Event Handling: Uses jQuery to trigger the accordion effect.

🎭 Dynamic Header Animation: The title changes color and size when a question is opened.

🌍 Responsive Design: Ensures usability across different screen sizes.

🛠 Technologies Used

HTML5: Structure of the webpage.

CSS3: Styling and layout.
